# MVP Scope

## MVP Goals
- Deliver coordination intelligence with minimal friction
- Provide clear, explainable insights
- Integrate with real developer workflows (initially GitHub + manual inputs)

## MVP Features
- Organizations and teams
- Projects with membership
- Tasks with dependencies
- Blocker detection and critical path insights
- Workload balancing suggestions
- Explainable AI recommendations
- Event log for audit trail
- Basic notifications (in-app)

## Explicitly Out of Scope (MVP)
- Enterprise roles and granular permissions
- Advanced billing and usage tiers
- Deep multi-tool integrations beyond GitHub
- Complex OKR or HR workflows

## MVP Milestones
1. **Foundation**: auth, orgs, teams, projects, tasks
2. **Coordination Core**: dependencies, blockers, workload view
3. **AI Explanations**: reasoning output with evidence
4. **Integration Lite**: GitHub issue/PR sync
5. **Feedback Loop**: user corrections to improve AI
